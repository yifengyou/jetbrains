# JetBrains全家桶 IDE 常用技巧

## 启用Vim

File->Settings->Plugins 安装IdeaVim插件

![20210712_212220_63](image/20210712_212220_63.png)

File->Settings->Keymap 选择**适合**自己的开关，用于Vim模式和Idea模式快速切换


![20210712_212319_94](image/20210712_212319_94.png)
